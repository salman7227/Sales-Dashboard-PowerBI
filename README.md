# Product Sales Data Analysis | Power BI

## Overview
This Power BI project analyzes product sales performance using interactive dashboards. It helps identify sales trends, top-performing products, profit margins, and regional performance to support business decision-making.

## Objectives
- Analyze overall sales performance
- Track profit and revenue
- Identify top-selling products
- Compare regional sales
- Monitor monthly sales trends

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV

## Dashboard Features
- KPI Cards (Sales, Profit, Orders)
- Monthly Sales Trend
- Product Category Analysis
- Region-wise Sales
- Top Products
- Interactive Filters & Slicers
